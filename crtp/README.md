# CRTP

The Curiously Recurring Template Pattern (CRTP) is a C++ idiom .

- using crtp for mixins
 
```shell
gcc ./mixin.cpp   && ./a.exe 
```

- using crtp for Static polymorphism

```shell
gcc ./static_interfaces.cpp   && ./a.exe 
```
